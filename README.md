
#  F.fetch
My personal configs to rice fastfecth `(skidmaxx)`

    To install and Use it
* Redirect/clone this repo to `~/.local/share`
* Type this command in the terminal to use it `fastfetch --config main.jsonc` change the file name in this command to use other configs and always make sure to be on the right directory

Only 2 configs are available planning to add more in the future
